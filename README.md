# Random User Generator

A Python desktop application that generates and displays random user profiles using the Random User API.

## Features

- 🎲 Generate random user profiles with one click
- 👤 Display user information including:
  - Full name with title
  - Gender
  - Email address
  - Phone number
  - Location (city, state, country)
  - Date of birth with age
  - Nationality
- 🖼️ Show profile avatar images
- 💾 Save user data to JSON files
- 🎨 Clean and modern user interface

## Technologies Used

- Python 3.7+
- Tkinter (GUI framework)
- PIL/Pillow (Image processing)
- Requests (API communication)
- Random User API
